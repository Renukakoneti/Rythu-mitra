# 📱 Rythu Mitra App

The premiere mobile interface for common-sense precision agriculture. This React Native application (built with Expo) provides farmers with real-time field intelligence, hardware diagnostics, and biological growth tracking at their fingertips.

## ✨ Features
- **Modern Dashboard**: Unified view of device health and climate trends.
- **Smart Telemetry**: Live visualization of Soil Moisture, Temperature, CO2, and Light.
- **Field Analytics**: Advanced biological calculations like VPD and Photosynthetic index.
- **Account Recovery**: Secure phone-based verification and access key recovery.
- **Multi-Node Support**: Manage an entire fleet of IoT nodes from one account.

## 🎨 UI/UX Design
- **Premium Aesthetics**: Dark-mode optimized, slate-and-emerald color palette.
- **Lucide Icons**: Crisp, diagnostic-grade visual language.
- **Responsive Layouts**: Built with NativeWind (TailwindCSS) for perfect scaling.

## 🚀 Tech Stack
- **Framework**: React Native (Expo)
- **Styling**: NativeWind
- **API Communication**: Axios
- **Secure Storage**: Expo SecureStore
- **Icons**: Lucide React Native
