# 🌾 Rythu Mitra Backend

The mission-critical intelligence hub for the Rythu Mitra smart farming ecosystem. This Node.js/Express server handles real-time telemetry ingestion, user authentication, and advanced growth analytics.

## 🚀 Core Features
- **JWT Authentication**: Secure login and session management.
- **IoT Data Pipeline**: High-performance ingestion for hardware sensor streams.
- **Growth Environment Analytics**: Real-time VPD (Vapor Pressure Deficit) and growth index calculations.
- **Alert System**: Context-aware notification management for field events.

## 🛠️ Technology Stack
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MongoDB (Mongoose ODM)
- **Security**: JSON Web Tokens (JWT) & Bcrypt hashing

## 📁 Key Directories
- `/routes`: API endpoints (Auth, Devices, Telemetry, Dashboard, IoT).
- `/models`: Mongoose schemas for Users, Devices, and Sensor Data.
- `/middleware`: Security firewalls (Auth & IoT verification).
- `/config`: Token generation and API documentation.
